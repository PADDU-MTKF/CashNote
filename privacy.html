Privacy Policy

Last updated: March 2026

Cash Note is designed with a strong focus on user privacy. This Privacy Policy explains how the app handles data and permissions.

1. Data Collection

Cash Note does not collect, store, or transmit any personal data to external servers. The application is completely serverless and operates locally on the user's device. All financial information entered in the app remains stored locally on the device.

2. Server Usage

Cash Note does not use any backend servers. No user data is sent to, stored on, or processed by any developer-controlled server. The application functions completely offline. Internet access is required only if the user enables optional Google Drive backup.

3. Google Account Access

Signing into a Google account is optional. Users may use the app fully without signing in. If a user chooses to sign in, the authentication is handled through Google Sign-In. This is used only to allow the user to store backups in their own Google Drive.

4. Google Drive Backup

Cash Note provides an optional feature that allows users to back up their app data to their personal Google Drive account. Important points:

  • Backup is optional
  • Data is stored only in the user's Google Drive
  • The developer does not have access to this data
  • No other files in Google Drive are accessed

Google Drive is used only for:

  • Creating backup files
  • Restoring backup files

5. Local Storage Access

The app may request storage permission to create local backups. Storage access is limited to a dedicated application folder used for storing backup files. Cash Note does not access other files on the user's device.

6. Notifications

Notification permission is requested to provide:

  • Reminder notifications
  • Automatic backup notifications

Notifications can be disabled by the user at any time through the app settings or device settings.

7. Google Calendar Integration

Google Calendar access is optional and only used if the user enables calendar reminders. This permission allows the app to:

  • Add reminder events
  • Remove reminder events

No other calendar data is accessed. Users can disable this feature anytime.

8. Offline Operation

Cash Note is designed to function completely offline. Internet access is only required if the user chooses to enable cloud backup using Google Drive.

9. Third-Party Services

Cash Note uses the following Google services:

  • Google Sign-In
  • Google Drive API
  • Google Calendar API

These services operate under Google's own privacy policies.

10. User Control

Users have full control over their data. Users can:

  • Delete their data
  • Disable cloud backup
  • Disable calendar integration
  • Disable notifications

11. Children's Privacy

Cash Note does not knowingly collect personal information from children.

12. Changes to This Policy

This privacy policy may be updated in the future. Updates will be reflected on this page.
